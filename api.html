Developer API

Fast endpoints for transaction scoring, event streaming, and model feedback.

POST

/v1/score

Returns a risk score and explanation details for each transaction batch.

POST

/v1/events

Sends model events and decision metadata to your monitoring tools.

PATCH

/v1/feedback

Submit analyst outcomes so the model can improve accuracy and reduce false positives over time.
